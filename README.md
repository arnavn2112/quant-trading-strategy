# Quant Trading Strategy Backtest

## Overview
This project implements a simple moving average crossover trading strategy using Python.

## Strategy
- Buy when 20-day moving average crosses above 50-day
- Sell when 20-day moving average crosses below 50-day

## Tools Used
- Python
- pandas
- matplotlib
- yfinance

## Results
The strategy is compared against the overall market performance.
